Question-3: What is event delegation in the DOM?

Ans:Event Delegation is a technique in JavaScript where a single parent element handles events for multiple child elements, even if they are added dynamically

Question-4: What is event bubbling in the DOM?

An:Event Bubbling is a mechanism where an event triggered on a child element propagates (bubbles) up to its parent and ancestors in the DOM
